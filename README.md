# Cattle_Cow_Collar
EGR4820/EGR4830 Senior Project  
California State Polytechnic University, Pomona  
Electrical and Computer Engineering Department  

This repository contains code for a smart collar mounted onto a cow to track metrics such as body temperature and location and display it on an application that shows the location of the cow as long as its metrics.
